# Quickstart — UI Expense

1) Install deps
```bash
npm ci
```

2) Run unit tests (expense UI)
```bash
npm run test
```

3) Run Playwright smoke
```bash
npm run test:e2e
```

4) Open Coverage Index (after CI)
- Download `review-packet` from PR run
- Open `review-artifacts/index.html` → UI Coverage → UI‑expense
